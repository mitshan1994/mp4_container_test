# Container Tests

This is a project for learning `mov`/`mp4`. It parses important boxes when extracting video and audio data.

Currently, `AVC` and `HEVC` are supported. `AAC` is supported.

Other tested formats:

- flv
- mkv
- mpeg2 ts/ps
- rtmp
